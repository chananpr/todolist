Workflow

ทีมใช้ site นี้ยังไง

1

Design review

เปิด pages.html ระหว่างประชุม เพื่อ review layout และจัด priority หน้า

2

Implementation pairing

นัก dev เปิด components.html ข้าง editor — copy markup/class ไปใช้ได้เลย

3

Onboard คนใหม่

ให้สมาชิกใหม่อ่าน sitemap.html และ graphs.html เพื่อเข้าใจระบบใน 30 นาที

Quick start

เปิด site นี้ให้ทีม

รันบนเครื่องตัวเอง
# จาก root project
npm run design

# หรือ
cd design && python3 -m http.server 8080
# เปิด http://localhost:8080
แชร์ให้ทีม
# ใช้ ngrok / cloudflared / tailscale
cloudflared tunnel --url http://localhost:8080

# หรือ deploy เป็น static site
# (ทุกไฟล์ใน design/ คือ static)
