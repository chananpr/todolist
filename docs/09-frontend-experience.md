# 09 Frontend Experience

## Experience direction
The client should feel like a modern operations command center rather than a plain CRUD admin panel.

## Visual principles
- strong typography with a non-default family
- deep navy and graphite base with copper and teal highlights
- layered gradients and glows for depth
- subtle motion for hierarchy and focus, not decorative noise
- responsive layouts for desktop and tablet first, mobile supported

## Core pages
- login
- dashboard overview
- my tasks
- today work
- projects list
- project detail
- kanban board
- task detail drawer
- employee directory
- roles and permissions
- KPI dashboard
- activity center
- AI planner preview and apply screen

## Core UI components
- app shell with sidebar and command area
- animated stat cards
- kanban board with drag and drop
- task drawer with comments, subtasks, costs, and files
- AI recommendation panel
- KPI widgets and trend strips
- recent activity timeline
- calendar and schedule panel

## Animation guidance
- use page enter transitions and staggered card reveals
- use hover elevation and soft light sweeps on interactive cards
- use drag feedback states that clearly indicate target drop lane
- keep motion under 250ms for routine interactions
